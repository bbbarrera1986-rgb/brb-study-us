      Help & Error Diagnostics
      
    

    


      

        
🔑 Invalid API Key

        
SYMPTOMError 401 or 403 when generating.
SOLUTIONGemini key starts with AIza. OpenAI: sk-. Claude: sk-ant-. No spaces.

      


      

        
📊 API Quota Exceeded

        
SYMPTOMError 429 — "quota exceeded" or "limit: 0".
SOLUTIONGemini free: wait 1 minute. If limit shows 0, the model may not be available in your region — try a different provider.

      


      

        
✂️ Output Cut Off / Incomplete Material

        
SYMPTOMGenerated text stops abruptly mid-sentence.
SOLUTIONTry a simpler study type (Summary or Concept) or reduce context length. Use Chat AI tab to ask for the missing section.

      


      

        
⏰ License Expired or Device Not Authorized

        
SOLUTIONContact BRB Estrategia & Procesos — bbbarrera1986@gmail.com · +54 297 517-6351.

      


      

        
💾 Daily Limit of 10 Generations Reached

        
SOLUTIONResets at midnight. Your history and saved sessions are not lost. Use Chat AI tab to continue working without using a generation.

      


      

        
🌐 Connection Error / No Internet

        
SOLUTIONCheck your internet. Temporarily disable ad blockers and VPN. SAVI and Analogies use separate API calls — if they fail, the main material is still saved.

      


      

        
📄 Attached File Does Not Process

        
SOLUTIONScanned PDFs (images) are not readable. Use PDF with selectable text or Word .docx format.

      


      

        
📋 SAVI or Analogies Show Error

        
SOLUTIONThey are separate API calls. The main material is already saved. Regenerate by generating the full material again with the toggle ON.

      


      

        
💻 Blank Screen on Load

        
SOLUTIONCtrl+Shift+R to force reload. If it persists, open F12 → Console, copy the error and contact BRB.
